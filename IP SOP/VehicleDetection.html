We calculate the threshold using Otsu Threshold, and apply it on the image.

Canny Edge Detection¶

Using the binary image usr use Canny Edge Detection technique to fetch the edge boundaries in the image.

Smoothing The Edges¶

We now try to use GaussianBlur to connect and smoothen the edge for better hole filling.

Filling Holes¶

The vehicle holes by canny edge detection are filled as 255 value. This is achieved using morphological close operation.

In [7]:
# Defining the kernal for the Morphological Close Operation:
kernel = np.ones((9,9),np.uint8)
closing = cv2.morphologyEx(smooth, cv2.MORPH_CLOSE, kernel)


# Plotting the hole filled image: 
plt.imshow(closing,cmap = 'gray')
